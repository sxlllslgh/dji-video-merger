# 大疆视频拼接器

### 把多段视频拼接整理成一条完整时间线，并为成片叠加清晰、专业、可拖拽的运动数据仪表盘

一个面向终端用户的在线网页工具，适合用于 DJI 视频拼接、飞行数据可视化、GPS 轨迹叠加、速度海拔仪表盘生成，以及透明背景仪表盘序列导出。

`多段视频拼接` `整体预览` `轨迹叠加` `速度/海拔可视化` `透明背景导出`

## 核心特色

| 模块 | 功能 |
| --- | --- |
| **视频拼接** | 多段素材按最终顺序统一处理，适合由于自动分段或手动启停导致的多段视频拼接 |
| **运动数据拼接** | 解决多段视频导致的运动数据不连续和轨迹、里程分段 |
| 整体预览 | 先确认视频画面、时间轴和仪表盘效果，再导出 |
| 仪表盘布局 | 速度、海拔、里程、日期时间、轨迹、加速度等组件可自由摆放 |
| 透明导出 | 导出透明背景 `SVG` / `PNG` 序列，直接叠加到正式工程 |
| 路线展示 | 在画面中叠加完整轨迹与当前位置，提升空间感与信息量 |
| **数据补偿** | 海拔丢失补齐、速度丢失估算，减少断点造成的跳变感 |

## 你可以在页面里完成什么

1. 选择多段航拍视频文件。
2. 拖动排序，确定拼接顺序。
3. 生成整体预览，确认完整效果。
4. 勾选要显示的仪表盘项目。
5. 在预览区拖动仪表盘位置。
6. 选择导出透明背景仪表盘序列，或同时导出合并视频。

## 导出结果适合做什么

- 作为透明叠加层放入 Premiere Pro
- 作为透明素材序列导入 Final Cut Pro
- 作为动态图形辅助层进入 DaVinci Resolve
- 作为二次包装元素用于 After Effects 合成
