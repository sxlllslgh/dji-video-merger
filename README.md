## ✨ 核心特色

| 模块 | 功能 | 亮点 |
| :--- | :--- | :---: |
| **🎬 视频拼接** | 多段素材按最终顺序统一处理 | 自动分段 / 手动启停均适用 |
| **📊 运动数据拼接** | 解决多段视频导致的数据不连续 | 轨迹 + 里程无缝衔接 |
| **🖥️ 整体预览** | 先确认画面、时间轴和仪表盘效果，再导出 | 所见即所得 |
| **🧩 仪表盘布局** | 速度、海拔、里程、日期时间、轨迹、加速度等组件 | 自由拖拽摆放 |
| **🎨 透明导出** | 导出透明背景 SVG / PNG 序列 | 直接叠加到正式工程 |
| **🗺️ 路线展示** | 画面中叠加完整轨迹与当前位置 | 提升空间感与信息量 |
| **🔧 数据补偿** | 海拔丢失补齐、速度丢失估算 | 减少断点跳变感 |

## 📋 工作流程

```
选择视频 → 拖动排序 → 整体预览 → 配置仪表盘 → 拖拽定位 → 导出
```

1. **选择** 多段航拍视频文件
2. **拖动排序**，确定拼接顺序
3. **生成整体预览**，确认完整效果
4. **勾选** 要显示的仪表盘项目
5. 在预览区 **拖动** 仪表盘位置
6. **导出** 透明背景仪表盘序列，或同时导出合并视频
